PROJECT DESCRIPTION 
The digital literacy project aims to build the necessary awareness and skills required to effectively interact in the digital world. The digital literacy project is based on the following activities:
- Designing an infographic to create awareness
- Developing a professional digital portfolio
- Learning about coding and collaboration tools
- Learning the etiquettes of sending and receiving emails
- Learning about cybercrimes and the prevention of such crimes

The digital literacy project aims to improve the communication and technical skills of the individual.
PROJECT STRUCTURE 
Task 1: Infographic
Designed a digital literacy awareness infographic using Canva. Topics covered include safe internet usage and digital tools.

Task 2: Digital Portfolio
Created professional profiles on platforms such as GitHub and LinkedIn to develop an online presence.
 
Task 3: Platforms
Explored coding platforms and created a Google Form quiz for digital literacy awareness.
 
Task 4: Email Etiquette
Created professional emails and a checklist for responsible social media usage.
 
Task 5: Cybercrime Awareness
Created a case study on cybercrime and a checklist for responsible online usage.
